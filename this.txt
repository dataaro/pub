You can create files using any tool, but Visual Studio Code is recommended for consistency across operating systems.
After creating a file (e.g., example.txt), you can add it to the staging area with the command git add followed by the file name. This prepares the file for the next commit.
These steps are foundational for managing your projects with Git, especially in collaborative environments like the one you're working in.
